# CDS 知识库

本目录记录 CDS 组件开发过程中积累的经验、模式和踩坑记录。

## 文件命名

- `pattern-{name}.md` — 可复用的实现模式
- `pitfall-{name}.md` — 踩坑记录和解法
- `decision-{name}.md` — 组件设计决策（为什么这样做）

## 当前条目

- `pattern-portal-interaction.md` — createPortal 与外部点击关闭的交互模式
- `pattern-tailwind-v4-tokens.md` — Tailwind v4 token 引用语法
- `pattern-table-column-sizing.md` — CdsTable 列宽控制（1% 收缩技巧、frozen left/right 区别）
- `pitfall-frozen-column-zindex.md` — CdsTable frozen 列内弹出层必须 z-200+
